# Module 01 – Understanding Computers & Programming

## 🎯 Objective

By the end of this module, you should understand how a computer runs a program and become familiar with some of the basic terms used in programming.

---

# 📚 Concepts to Explore

Learn about the following concepts using any resource you prefer (books, videos, blogs, official documentation, AI, etc.).

## Computer
A machine that takes input, processes it, and produces output.

## Hardware
The physical components of a computer that you can see and touch.

## Software
Programs and applications that tell the computer what to do.

## Operating System (OS)
Software that manages the computer's hardware and allows applications to run.

## Programming
The process of giving instructions to a computer to solve a problem.

## Programming Language
A language that humans use to write programs, such as C, C++, Java, or Python.

## Source Code
The code written by a programmer.

## Machine Code
The binary instructions (0s and 1s) that the computer understands.

## Compiler
A program that converts source code into machine code before execution.

## Interpreter
A program that executes code one instruction at a time.
